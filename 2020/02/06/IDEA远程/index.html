IDEA远程debug

Posted on 2020-02-06

一. 远程服务器:

  1. 打开服务器的5005端口,注意配置防火墙;
  2. 在远程服务器上启动Java程序的时候, 添加参数: -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005
    比如: java -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005 -jar starter-0.0.1-SNAPSHOT.jar

二. 本地:

  1. 要确保本地的代码与服务器的jar包的代码一致.
  2. 在本地的IDEA设置以下配置.

三. 启动:

先使用命令运行远程服务器上的Java程序, 然后在IDEA使用debug模式启动刚才的配置.

在IDEA的Console栏中会出现这行:
Connected to the target VM, address: 'xxx.xxx.xxx.xxx:5005', transport: 'socket'
说明成功了进入远程debug.

安装字体

(兄弟们, 不要跟我提啥连字符 和 各种关于字体的情怀, 连字符我看着不舒服, 就是这么简单, 个人习惯;)

以下是我的推荐:
写代码使用 Bitstream Vera Sans Mono
第二字体使用 Sarasa UI SC (如果有中文字符, 如字符串或者中文注释, IDEA会去使用第二字体)
控制台字体使用 DejaVu Sans Mono for Powerline

(想要情怀加成的, 自己去了解这三种字体, 这只是暂时使用的字体, 如果我发现了更好的字体,我会更新的)

Bitstream Vera Sans Mono
Sarasa UI SC
DejaVu Sans Mono for Powerline

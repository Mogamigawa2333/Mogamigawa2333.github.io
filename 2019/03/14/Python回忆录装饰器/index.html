闭包(Closure)

闭包是由函数与其相关引用环境组成的实体

闭包的成立条件:

  1. Python中函数可以嵌套定义
  2. 按照变量的作用域, 内部函数可以访问外部函数的变量
  3. 函数可以当作值来返回

闭包的目的: 能够读取其他函数的内部变量;

在上面的代码中: 当返回函数inner的时候, 函数inner与外部函数outer的一个内部变量相关联, outer执行后生命周期就停止了,但是由于inner引用了outer的内部变量, inner与其相关的引用环境组成了实体;

*args, **kwargs

*args, **kwargs 它们两个是为了解决业务逻辑函数需要参数的问题 (在我们的例子中是foo函数)

Merge远程分支:
A merge B 是把A中的改动放到B分支上,B merge A是把B中的改动merge到A中; —>
例如把C分支上的改动移到分支master上,可以这样做:(C merge master)
1.转到C分支上,然后更新C最新更新;
2.然后在Local Branches中选择master分支,选择merge,然后再选择git push.
这样就把远程C merge到 master上, 并会提示: Merged C to master;
3.回复邮件;

git版本回退:
回退的是本地的commit提交的文件;
HEAD指向的版本就是当前版本,因此,Git允许我们在版本的历史之间穿梭,使用命令git reset –hard commit_id
穿梭前,用git log可以查看提交历史,以便确定要回退到哪个版本
要重返未来,用git reflog查看命令历史,以便确定要回到未来的哪个版本

commit id (版本号)是一个SHA1计算出来的一个非常大的数字,用十六进制表示,而且每个人的看到的commit id不一样,以自己的为准;

Dockerfile 语法

docker images

docker run -d -p <宿主机的端口>:8080 <镜像的id或者名字+tags>

docker exec -it <容器的id或者镜像的名字> /bin/bash

exit 或者 Ctrl+P+Q

docker stop <容器id或者容器的名字>

docker kill <容器id或者容器的名字>

1.Authentication 身份认证;
authentic：真实可信的
【结构分析】： authentic = aut(自己) + hent(行为人) + ic(形容词后缀) →自主行为人所说的或所做的→并非屈打成招的→真实可信的
【词源解释】： aut <-希腊语 autos (自己); hent <-希腊语 hentes (行为人)
【衍生词】： authenticity (真实性); authenticate (鉴定); authentication (证明)

2.Authorization 授权
authorize
词根词缀： auth ( -auct- )增加->创造->权威 + -or 名词词尾 , 行为 + -ize 动词词尾

3.realm：王国，国度;界，领域，范围
拉丁语中动词 regere (统治)派生的 regimen (政府系统) , 嫁接了古法语 reiel (王室的 , 国王的) , 而成为英语 realm .

  • reg- 统治
    【同源词】： regime, regimen, regiment

Haskell是强类型,静态隐式类型;

  • 强类型: 不允许类型转换,比如JS中的’666’/2=333
  • 编译时就知道变量类型的是静态类型；运行时才知道一个变量类型的叫做动态类型
  • 静态类型中,通过类型推导确定变量类型的是隐式类型; 在声明变量时说明变量类型的是显式类型,例如 int age = 18;
